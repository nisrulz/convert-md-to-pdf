# Convert Markdown to PDF


## Installation

Install [Puppeteer](https://github.com/puppeteer/puppeteer): 

```sh
npm i  puppeteer
```


# Render to PDF

Pass the HTML file as cmdline argument before running the `renderToPdf.js` script.

```sh
node renderToPdf.js integration_guide.html
```
